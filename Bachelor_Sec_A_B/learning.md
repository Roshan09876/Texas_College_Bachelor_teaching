# User Register 

## For Normal User
  1. Fill up the form 
  2. click register button

## For Developer 
  1. Destructure form data
  2. check for the validation
  3. check for the unique email 
  4. hashing the password
  5. generate token
  6. store data 
  7. message via status code. 
      - 200 (success...)
      - 201 (created... new data)
      - 400 (normal error msg)
      - 404 (...not found)
      - 500 (Internal Server error)
